# 双CRSF接收机冗余备份系统设计文档

## 系统概述

### 核心功能
1. **双路CRSF信号解析** - 通过两个UART接收两个ELRS接收机的CRSF协议数据
2. **信号质量监控** - 实时监控RSSI、Link Quality等参数
3. **智能切换** - 根据信号质量自动选择主接收机或切换至备用接收机
4. **10路PWM输出** - 将CRSF信号转换为标准PWM信号输出给飞控

### 硬件平台
- **主控芯片**: STM32F407VGT6 或 STM32F103ZET6
- **串口**: UART1 (RX1) + UART2 (RX2) 用于CRSF接收
- **PWM输出**: Timer2/Timer3/Timer4 产生9路PWM信号
- **波特率**: 420K baud (CRSF标准)

## CRSF帧结构解析

### 基本帧格式
```
Sync(1) | Length(1) | Type(1) | Payload(n) | CRC(1)
  0xC8  |     n     |  type   |  Payload   | crc8
```

### 关键帧类型
- **0x14** - Link Statistics (主要监控RSSI和LinkQ)
- **0x16** - RC Channels Packed (16路PWM通道打包数据)

## PWM输出配置

### Timer分配方案 (STM32F407)
- **Timer2 CH1-3** - PWM通道1,2,3 (PA0, PA1, PA2)
- **Timer3 CH1-3** - PWM通道4,5,6 (PA6, PA7, PB0)
- **Timer4 CH1-3** - PWM通道7,8,9 (PB6, PB7, PB8)

### PWM参数
- 频率: 50Hz (遥控标准)
- 分辨率: 1-2000µs (1000=1500µs中位)
- 精度: 1µs

## 切换策略

### 主从选择规则
1. 初始状态: 两路都接收时优先主路
2. 主路RSSI > -85dBm 且 LinkQ > 50% 时使用主路
3. 主路RSSI < -95dBm 或 LinkQ < 30% 时切换至备用路
4. 两路都丢信号时输出安全PWM (1000µs)

### 故障检测
- 无CRSF帧超过100ms: 认为信号丢失
- RSSI严重恶化触发切换
- CRC校验失败不使用该帧

## 调试和监控

### 输出信息
- UART3 调试口输出系统状态
- 每100ms输出一次RSSI和LinkQ
- 切换事件记录

|------|------|------|
